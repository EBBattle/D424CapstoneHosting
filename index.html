Installation and Usage Instructions

Smart Planner & Budget Coach is an Android mobile application designed to help users plan vacations, track excursions and expenses, and manage their overall budget. The application operates entirely offline using local data storage and does not require an internet connection for normal use.

System Requirements

  • Android device running Android 8.0 (API level 26) or higher
  • Minimum screen resolution: 720p
  • No internet connection required for normal operation
  • Application installed via APK deployment

Obtaining the APK

The application APK can be obtained using one of the following methods:

  • Download the APK directly from this deployment page on the Android device
  • Transfer the APK to the device using USB, email attachment, or cloud storage

Allowing Installation from Unknown Sources

Android blocks APK installations by default unless the user explicitly allows installations from unknown sources. To enable this:

  1. Open Settings
  2. Select Apps or Security / Privacy
  3. Locate Special App Access
  4. Select Install Unknown Apps
  5. Choose the app used to open the APK (Chrome, Files, Drive, etc.)
  6. Enable Allow from this source

Installing the APK

  1. Open the downloaded APK file
  2. Review the installation prompt
  3. Tap Install
  4. Wait for Android to complete the installation
  5. Tap Open to launch immediately or Done to exit

Launching and Using the Application

After installation, the application can be launched by tapping the Smart Planner & Budget Coach icon from the device home screen or app drawer. Users can then create vacations, add excursions and expenses, and track total costs against their defined budget.
